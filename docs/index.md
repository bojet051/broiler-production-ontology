# Industrial Broiler Production Ontology

Welcome to the **Industrial Broiler Production Ontology** (IBPO) documentation.

## Overview

The IBPO is a comprehensive systems ontology for industrial broiler production that integrates knowledge from:
- **Biology** - avian physiology, genetics, breed characteristics
- **Engineering** - infrastructure, environmental controls, monitoring systems
- **Veterinary Science** - health, diseases, welfare management
- **Economics** - production costs, resource utilization, performance metrics

## Key Features

- **Well-defined individuals** for production phases (Brooding, Growing, Pre-Processing), genders (Male, Female), and observable characteristics (Weight, Temperature)
- **Comprehensive relationships** between broilers, infrastructure, sensors, and observations
- **Strict cardinality constraints** ensuring data consistency
- **Deprecated properties** maintained for backward compatibility
- **Multi-format releases** (OWL, Turtle) for interoperability
